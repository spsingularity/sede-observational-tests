# sede-observational-tests

**SEDE against DESI DR2, Pantheon+ and Planck 2018: a Zero-Extra-Parameter Test**

Paper IV (data): full-likelihood comparison of SEDE vs LCDM through the mochi-class Boltzmann code in cobaya. DeltaDIC = -3.5; +2-3% fsigma8 and -7.6% ISW predictions; solver-pathology diagnosis and fix.

Part of the USC research program (SEDE / ECCG / APDM unification).

## Status
Pre-submission working repository, seeded 2026-07-17 from a private research hub;
full research history (including audit trails and
retractions) lives there.

## Layout
- `paper/` — manuscript sources, `refs.bib`, `build.sh`, and `paper/tools/` build machinery
- `src/` — the reproduction pipeline (MCMC, sims, results, checksums)
- `figures/` — generated figures
- `results/` — TIER result narratives
